# Agrego Web

Agrego Web is a Bun-powered React, TypeScript, Vite, PWA frontend for the Agrego digital cooperative platform with Convex backend schema support.

## Essentials

- Use Bun for package scripts and dependency management.
- Do not run `bun run dev` or `bun run build`; use `bun run lint` for verification unless the user explicitly permits another command.
- Read only the task-relevant instruction file before editing:
  - [Frontend conventions](docs/agents/frontend.md) for React, TypeScript, styling, and PWA work.
  - [Backend conventions](docs/agents/backend.md) for Convex schema and server-side work.
  - [Verification](docs/agents/verification.md) for checks that are allowed in this repo.
  - [Instruction maintenance](docs/agents/instruction-maintenance.md) when updating these agent instructions.
